# Life OS

A personal AI-assisted life operating system, built iteratively
and documented as a case study.

## What this is

A system for daily planning, task management, and intentional
time use — powered by Claude Code in V1, evolving toward an MCP
server and web dashboard.

## Version roadmap

- **V1** (current): Claude Code CLI + flat files + GitHub
- **V2** (planned): MCP server + Supabase + calendar integration
- **V3** (planned): React/Next.js web UI

Each version earns the next.

## Project structure

```
CLAUDE.md          # System prompt — drives Claude Code
ROADMAP.md         # Phased build plan
JOURNAL.md         # Case study entries
mcp-planning.md    # Emerging MCP tool patterns
preferences.md     # Confirmed user preferences
todos.md           # Active tasks
log.md             # Timestamped activity log
decisions/         # Architectural decision records
retrospectives/    # Weekly retrospectives
```

## Case study

This project is being documented as a case study in building a
personal AI life OS. The commit history, decision records, and
journal entries tell the story of how the system evolved through
actual use.
